# 📚 Clubinho do Livro

Bem-vindo ao **Clubinho do Livro**, uma plataforma criada para amantes da leitura descobrirem novos livros, compartilharem avaliações e explorarem recomendações da comunidade.

## 📖 Sobre o Projeto

O Clubinho do Livro é um site desenvolvido utilizando HTML e CSS com o objetivo de criar um ambiente simples e agradável para leitores. A plataforma permite visualizar livros, consultar avaliações feitas por usuários e acessar páginas de cadastro e login.

## 🚀 Funcionalidades

- Página inicial com destaque para livros.
- Catálogo de livros.
- Avaliações da comunidade.
- Sistema de login.
- Página de cadastro de usuários.
- Interface responsiva e moderna.

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
